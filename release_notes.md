## v0.4.2 업데이트 내역

### Store 안정화
* **Store 빌드 동작 정리** — Microsoft Store 빌드에서 직접 폴더 열기처럼 심사에 민감한 동작은 숨기고, 직접 다운로드 빌드에서는 기존 편의 기능을 유지했습니다.
* **제출 버전 재사용 방지** — 이미 제출한 Store 패키지 버전을 다시 빌드하거나 업로드하지 않도록 릴리스 잠금과 preflight 검사를 추가했습니다.
* **패키징 검증 강화** — Store용 MSIX 버전, manifest, SHA256 기록, 임시 산출물 부재를 한 번에 확인하는 검증 흐름을 정리했습니다.

### 공간 확보 추천 개선
* **추천 항목 설명 보강** — 다운로드, 미디어, 임시 파일, 캐시, 휴지통, 사용자 파일을 카테고리로 구분하고 다음 확인 단계를 함께 보여줍니다.
* **복구 안내 추가** — 개인 파일 삭제는 시스템 복원 지점으로 되돌릴 수 없다는 점을 명확히 안내합니다.
* **새로고침 실패 처리 개선** — 기존 결과가 있는 상태에서 재스캔이 실패해도 목록을 지우지 않고 재시도 안내를 표시합니다.

### 품질 및 릴리즈
* **Tauri 명령 검사 강화** — 생성된 바인딩 사용과 직접 `invoke(...)` 호출을 함께 검사해 누락된 백엔드 명령을 더 빨리 잡습니다.
* **후보 릴리스 검증 정리** — WinGet manifest가 아직 없는 후보 빌드는 placeholder 없이 검증하고, 실제 릴리스 전에는 strict 검사를 유지합니다.
* **0.4.2 배포 자산 제공** — Windows용 EXE/MSI 설치 파일을 최신 릴리스로 제공합니다.

### 🌍 언어별 앱 제목 및 버전 표시

* **버전 소스 일원화** — Tauri 설정 버전을 프론트엔드 fallback으로 주입해 브라우저 개발 모드와 Tauri 앱 모두 같은 버전을 표시합니다.
* **릴리즈 버전 정리** — 앱, npm 패키지, Cargo 패키지, Tauri 설정 버전을 `0.3.2`로 맞췄습니다.
