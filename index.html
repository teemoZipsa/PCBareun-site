CORE FEATURES

관리에 필요한 것만, 정확하게.

한 화면에서 시스템 상태를 보고, 다음 화면에서 바로 조치합니다. 군더더기 없이 12개 카테고리가 사이드바에 정리되어 있습니다.

임시 파일 · 캐시 정리

시스템 임시파일, 브라우저 캐시, AI 개발 캐시까지 카테고리별 스캔 후 선택적 삭제. 평균 1~2GB 즉시 회수.

레지스트리 정리

12개 카테고리에서 사용되지 않는 항목만 안전하게 검출. 위험도 라벨 표시 + 복원 지점 자동 생성.

중복 · 유사 이미지 검색

파일 해시 기반 정확한 중복 탐지, Perceptual Hash로 해상도가 달라도 유사 이미지 자동 그룹화.

보안 · 개인정보

텔레메트리 차단, DNS 검사, 노출된 API 키 자동 탐지, 개인정보 항목 일괄 정리.

디스크 · 시스템 모니터링

S.M.A.R.T. 기반 SSD/HDD 상태(NVMe 지원), CPU/RAM/GPU/VRAM 실시간 그래프와 가동 시간.

AI · 개발 환경 도구

대용량 AI 모델 스캔(.gguf, .safetensors), Ollama 관리, 포트 점유, PATH 정비, VHDX 압축까지.

SAFETY BY DESIGN

실수로도 시스템을
망가뜨리지 않습니다.

유틸리티의 진짜 가치는 “하면 안 될 것”을 알고 있는 데 있습니다. PC 바른은 위험 작업 전후로 여러 단계의 보호 장치를 거칩니다.

시스템 핵심 보호 목록

csrss, lsass, explorer, TrustedInstaller 등 11개 이상의 핵심 프로세스·서비스는 종료/삭제 시도 자체를 차단합니다.

위험 경로 차단

Windows, Program Files, EFI 등 보호 폴더와 UNC 네트워크 경로는 강제 삭제 대상에서 자동 제외됩니다.

복원 지점 권장

레지스트리 정리 전 Windows 복원 지점 생성을 안내. “위험도 높음” 항목은 기본 선택 해제 상태로 시작합니다.

커맨드 인젝션 방지

외부 URL 열기는 http/https/mailto 화이트리스트 + 셸 특수문자 차단. 모든 권한 작업은 사용자 확인을 거칩니다.

FAQ

자주 묻는 질문.

정말 무료인가요? 광고나 결제가 있나요?
네. 모든 기능을 무료로 사용할 수 있으며 광고도 없고 결제 유도도 없습니다. 다만 라이선스 정책에 따라 소스코드의 복사·수정·재배포는 금지되어 있습니다.
레지스트리를 건드려도 안전한가요?
사용되지 않는 항목만 카테고리별로 정리합니다. Windows 핵심 레지스트리는 건드리지 않으며, 작업 전 복원 지점 생성을 안내합니다. “위험도 높음” 항목은 기본 선택 해제 상태입니다.
다른 정리 도구와 무엇이 다른가요?
광고와 번들 설치가 없습니다. 또한 일반 사용자용 정리 기능 외에 AI 모델 스캐너, Ollama 관리, 포트 점유 확인, VHDX 압축처럼 개발자에게 실제로 필요한 도구가 포함되어 있습니다.
Mac이나 Linux도 지원하나요?
현재는 Windows 10(1809+) 및 Windows 11만 지원합니다. 기능 대부분이 Windows API와 레지스트리에 의존하므로 다른 OS 지원 계획은 없습니다.
자동 업데이트는 어떻게 동작하나요?
앱이 시작될 때 자체 업데이터가 새 버전을 확인하고, 사용자 승인을 받은 뒤 백그라운드에서 다운로드 및 설치합니다. 강제 업데이트는 하지 않습니다.
오픈소스인가요?
소스코드는 공개되어 있지 않으며, 라이선스에 따라 복사·수정·재배포가 금지됩니다. 단, 릴리즈 노트와 변경 사항은 모두 공개됩니다.
