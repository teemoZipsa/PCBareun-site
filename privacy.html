PC Bareun Privacy Policy

Last updated: 2026-06-18

개인정보 처리방침

PC Bareun은 Windows PC 관리 유틸리티입니다. 이 앱은 임시 파일 정리, 설치 프로그램 관리, 서비스 및 작업 스케줄러 관리, 하드웨어 상태 확인, 개발 환경 최적화 등의 기능을 제공합니다.

수집하는 정보

PC Bareun은 기본적으로 사용자의 개인정보를 외부 서버로 전송하거나 판매하지 않습니다.

앱 기능 수행을 위해 사용자의 기기 안에서 다음과 같은 정보를 읽을 수 있습니다.

  • 디스크, 폴더, 파일 이름 및 파일 크기
  • 설치된 프로그램 목록
  • Windows 서비스 및 작업 스케줄러 항목
  • 시스템 로그 또는 진단 정보
  • 브라우저 캐시, 임시 파일, 레지스트리 MRU 등 정리 대상 후보
  • 개발 도구, AI 모델 파일, 환경변수, PATH 정보

이 정보는 기능 실행과 화면 표시를 위해 로컬 기기에서 처리됩니다.

외부 전송

PC Bareun은 앱 업데이트 확인을 위해 GitHub Releases 또는 업데이트 프록시 서버에 접속할 수 있습니다. 이 과정에서 앱 버전 확인에 필요한 네트워크 요청이 발생할 수 있습니다.

PC Bareun은 별도의 분석 SDK, 광고 SDK, 사용자 추적 SDK를 포함하지 않습니다.

사용자가 선택한 작업

사용자가 직접 실행한 정리, 삭제, 비활성화, 최적화 작업은 사용자의 기기에 변경을 일으킬 수 있습니다. PC Bareun은 위험한 작업에 대해 경고와 확인 단계를 제공하며, 가능한 경우 복원 지점 또는 백업 안내를 제공합니다.

로그

PC Bareun은 오류 진단을 위해 로컬 로그를 저장할 수 있습니다. 로그는 사용자의 기기에 저장되며, 사용자가 직접 공유하지 않는 한 외부로 전송되지 않습니다.

기본 로그 위치: %LOCALAPPDATA%\com.pcbareun.app\logs\

문의

개인정보 또는 앱 지원 문의는 GitHub Issues를 통해 접수할 수 있습니다.

Privacy Policy

PC Bareun is a Windows PC management utility. It provides features such as temporary file cleanup, installed program management, Windows service and task scheduler management, hardware status checks, and development environment optimization.

Information Processed

PC Bareun does not sell personal information or send personal data to external servers by default.

To perform its features, the app may read the following information locally on the user's device:

  • Disk, folder, file name, and file size information
  • Installed program lists
  • Windows services and scheduled tasks
  • System logs or diagnostic information
  • Browser cache, temporary files, registry MRU entries, and other cleanup candidates
  • Development tools, AI model files, environment variables, and PATH information

This information is processed locally for app functionality and display.

Network Access

PC Bareun may connect to GitHub Releases or an update proxy server to check for app updates. This can create network requests required for version checks.

PC Bareun does not include analytics SDKs, advertising SDKs, or user tracking SDKs.

User-Initiated Actions

Cleanup, deletion, disabling, and optimization actions initiated by the user may change the user's device. PC Bareun provides warnings and confirmation steps for risky actions and provides restore point or backup guidance where possible.

Logs

PC Bareun may store local logs for troubleshooting. Logs are stored on the user's device and are not transmitted externally unless the user shares them manually.

Default log location: %LOCALAPPDATA%\com.pcbareun.app\logs\

Contact

Privacy or support inquiries can be submitted through GitHub Issues.
