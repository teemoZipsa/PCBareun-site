## v0.2.10 업데이트 내역 ✨

### 🔧 안정성 및 UI/UX 개선
* **대시보드 동시성 제어** — 새로고침 버튼 연타 시 무한 로딩(스피너)이 발생하는 문제를 방지하고 백그라운드 데이터 갱신 로직 안정화
* **시작 프로그램 관리자 개선** — 동일한 이름의 항목(예: 현재 사용자 / 모든 사용자)이 존재할 때 토글 상태가 꼬이는 문제 해결
* **시작 폴더 항목 UX 향상** — 시작 폴더에 위치한 항목 제어 시 비활성화 상태에 툴팁 안내를 추가하여 직관성 개선
* **시스템 정보 예외 처리** — GPU 센서 정보 로드 실패 시 이전 데이터가 남아있던 현상을 0/null로 올바르게 초기화하도록 수정
* **디스크 데이터 방어** — S.M.A.R.T. 정보가 비정상적으로 응답할 경우(Null 데이터) 대시보드 렌더링이 중단되는 문제 방어
* **Tailwind CSS 호환성** — 일부 환경에서 적용되지 않던 비표준 클래스(`h-4.5`)를 표준 픽셀 값(`h-[18px]`)으로 전환

### 🌍 다국어(i18n) 번역 무결성 강화
* **전용 번역 헬퍼(`tf`) 전면 도입** — 번역 파일(ko/en/ja)에 키가 누락되었을 경우를 완벽하게 대비하는 폴백(Fallback) 시스템 구현
* **설정 및 시작 프로그램 다국어 지원** — 하드코딩되어 있던 시스템 복원 지점 관리 및 시작 프로그램 위치(HKCU 등) 라벨에 다국어 동적 번역 적용

### ⚙️ 코드베이스 최적화
* **React Type Import 표준화** — 빌드 환경에 따라 발생할 수 있는 `React.ReactNode`, `React.ElementType` 네임스페이스 오류 선제적 조치
* **메모리 확보량 표시 정확도 향상** — VRAM/RAM 확보량 계산 시 음수가 표기될 수 있는 예외 케이스 차단

## v0.2.0 업데이트 내역 🛡️

### 🔒 보안 강화
* **URL 열기 보안** — `open_url` 커맨드에 스키마 화이트리스트(http/https/mailto) 및 셸 특수문자 차단 추가로 커맨드 인젝션 방지
* **강제 삭제 경로 보호** — UNC 네트워크 경로 완전 차단, 모든 드라이브에서 시스템 핵심 폴더(Windows, Program Files, EFI 등) 보호
* **프로세스 종료 안전장치** — 포트 모니터의 `kill_process`에서 시스템 핵심 프로세스(csrss, lsass, explorer 등) 종료 자동 차단
* **프로세스 부스터 보호 확대** — WerFault, TrustedInstaller, msiexec 등 11개 시스템 프로세스 추가 보호

### 🔧 안정성 개선
* **잔여파일 스캐너 정확도 향상** — 검색어 최소 길이 5자로 상향, "Microsoft", "Visual", "Adobe" 등 일반 단어 블랙리스트로 오탐 대폭 감소
* **업데이트 확인 정상화** — CSP에 `connect-src` 추가로 업데이터의 GitHub API 접근 허용
* **자동 업데이트 완전 지원** — `latest.json` 매니페스트 자동 생성으로 인앱 업데이트 체크 정상 작동

## v0.1.9 업데이트 내역 🚀

### 🔍 유사 이미지 검색 (신규 기능)
* Perceptual Hash(지각 해시) 알고리즘 — 해상도, 포맷, 압축률이 달라도 시각적으로 유사한 이미지를 자동 검출합니다.
* 유사도 임계값 조절 — 70%~100% 범위에서 검색 민감도를 슬라이더로 자유롭게 조절할 수 있습니다.
* 썸네일 그리드 UI — 유사 이미지 그룹을 썸네일로 한눈에 비교. "최고 품질" 뱃지로 보존할 파일을 즉시 확인.
* 자동 선택 — 가장 큰 파일(최고 품질)을 자동 보존하고 나머지를 선택하여 빠르게 정리.

### 👁️ 파일 미리보기 (신규 기능)
* 미리보기 모달 — 중복 파일 목록에서 👁️ 버튼을 클릭하면 이미지, 텍스트 파일을 바로 확인할 수 있습니다.
* 이미지 미리보기 — JPG, PNG, WebP, BMP, GIF 등 주요 포맷의 고해상도 미리보기 지원.
* 텍스트 미리보기 — TXT, JSON, CSV, 소스 코드 등 텍스트 파일의 내용을 500줄까지 미리 확인.
* 파일 정보 — 크기, 해상도, 수정일을 모달 하단에 한눈에 표시.

### 🛠️ 버그 픽스 및 안정성 강화
* 레지스트리 스캔 오류 수정 — 한글 Windows에서 PowerShell 스크립트 인코딩(UTF-8 BOM) 문제 해결.
* TypeScript 타입 에러 수정 — ProcessBoosterPage onClick 타입 불일치, TempCleanerPage 아이콘 타입 추론 오류 해결.
* PATH 관리 기능 강화 — 환경변수 PATH에서 문제 경로를 즉시 제거하는 기능 추가.
* 임시 파일 정리 UI — 개인정보 항목에도 적절한 아이콘 표시 추가.
