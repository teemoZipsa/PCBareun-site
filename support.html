PC Bareun Support Contact Information

Bug reports, installation issues, feature requests, and privacy questions are handled through GitHub Issues. Include the app version, Windows version, and reproduction steps so the issue can be reviewed efficiently.

Before posting a public issue

GitHub Issues may be public. Before posting, remove or hide sensitive details such as screenshots, logs, file paths, user names, document names, account information, and license information.

Install And Update

  • Microsoft Store builds of PC Bareun are updated through Microsoft Store.
  • If a Store update is not visible, open the Microsoft Store library and check for updates.
  • For GitHub or direct-install builds, check the latest release on the distribution page.
  • If possible, confirm that the issue still occurs on the latest version before filing a report.

Local Logs

For troubleshooting, PC Bareun local logs may be available at:

%LOCALAPPDATA%\com.pcbareun.app\logs\

Logs are stored on the user's device and are not transmitted unless the user chooses to attach or share them.

Privacy Questions

Privacy details are available in the privacy policy. Privacy inquiries can also be submitted through the same GitHub Issues support path.

설치와 업데이트

  • Microsoft Store에서 설치한 PC Bareun은 Microsoft Store를 통해 업데이트됩니다.
  • Store 업데이트가 보이지 않으면 Microsoft Store의 라이브러리 화면에서 업데이트 확인을 실행해 주세요.
  • GitHub 또는 직접 설치 버전은 배포 페이지의 최신 릴리스를 확인해 주세요.
  • 같은 문제를 신고하기 전에 가능한 경우 최신 버전에서도 문제가 반복되는지 확인해 주세요.

로컬 로그

오류 진단이 필요한 경우 PC Bareun의 로컬 로그가 아래 위치에 저장되어 있을 수 있습니다.

%LOCALAPPDATA%\com.pcbareun.app\logs\

로그는 사용자의 기기에만 저장되며, 사용자가 직접 첨부하지 않는 한 외부로 전송되지 않습니다.

개인정보 문의

개인정보 처리 기준은 개인정보처리방침에서 확인할 수 있습니다. 개인정보 관련 문의도 같은 GitHub Issues 경로로 접수할 수 있습니다.

サポート連絡先情報 (日本語)

不具合報告、インストールに関する問題、機能要望、プライバシーに関するお問い合わせは、上記のサポート URL から GitHub Issue として送信してください。 GitHub Issues は公開される場合があります。投稿前に、ファイル内容、アカウント情報、ユーザー名、ファイルパス、スクリーンショットなどの機密情報を削除または伏せてください。

含めると役立つ情報

  • PC Bareun のアプリバージョン
  • インストール元: Microsoft Store、GitHub Release、直接インストール
  • Windows のバージョンとビルド番号
  • 問題が発生した機能名
  • 問題を再現する手順
  • エラーメッセージ、スクリーンショット、または関連するローカルログの抜粋

インストールと更新

Microsoft Store 版は Microsoft Store を通じて更新されます。GitHub または直接インストール版を利用している場合は、最新のリリースページから更新してください。 可能であれば、問い合わせ前に最新バージョンでも同じ問題が発生するか確認してください。

ローカルログ

トラブルシューティング用のローカルログは、次の場所に保存されている場合があります。

%LOCALAPPDATA%\com.pcbareun.app\logs\

ログはユーザーの端末に保存され、ユーザーが共有しない限り外部へ送信されません。
