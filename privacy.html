PC Bareun Privacy Policy

PC Bareun is a Windows desktop utility for PC cleanup, diagnostics, and local system management. Cleanup and diagnostic data is processed locally by default. The app does not collect data for advertising or user tracking.

English 한국어 日本語
Local-first processing Scan, cleanup, and diagnostic results are calculated on the user's device for app functionality.
No ads or tracking PC Bareun does not include advertising SDKs, analytics SDKs, or user tracking SDKs.
User-confirmed actions Deletion and system changes run only for items selected and confirmed by the user.

1. Scope

This privacy policy applies to the PC Bareun Windows app and its public Microsoft Store listing. PC Bareun provides PC management features such as temporary file cleanup, duplicate file detection, disk health checks, installed program review, service and startup item review, BSOD analysis, and network diagnostics.

2. Core Principles

  • PC Bareun does not require a user account or sign-in.
  • PC Bareun does not collect data for advertising, behavioral tracking, or marketing profiling.
  • File deletion, program removal, and system-setting changes run only after the user selects and confirms the action.
  • Paths, file names, screenshots, and logs may contain personal information, so users should review them before sharing.

4. Local Storage

PC Bareun may store app settings, language and theme preferences, recent app state, feature results, and troubleshooting logs on the user's device. Local logs may include feature names, error messages, file paths, or file names.

Default log location: %LOCALAPPDATA%\com.pcbareun.app\logs\

This information is used to diagnose app issues and provide user-requested features reliably.

5. Network Requests

PC Bareun does not automatically upload cleanup results, file lists, or diagnostic results to external servers. Network requests may occur only when required for user-initiated actions such as:

  • Opening the website, support page, or GitHub links
  • Running network diagnostics or speed tests
  • Manually checking for updates in non-Microsoft Store direct-distribution builds

Microsoft Store builds are updated through Microsoft Store.

6. Third-Party Sharing And Sale

PC Bareun does not sell personal information. It also does not provide file contents, cleanup results, or diagnostic results to advertising companies or analytics services. If a user posts a GitHub Issue or manually attaches logs, that information is shared by the user's own action.

7. User Controls

  • Users can review and choose cleanup or deletion targets before applying changes.
  • Users can remove sensitive paths, file names, screenshots, or log content before sharing anything publicly.
  • Users can remove local auxiliary data by uninstalling the app or deleting local settings and logs.
  • Privacy inquiries can be submitted through the support path below.
Contact: https://github.com/teemoZipsa/PCBareun/issues

8. Changes

This privacy policy may be updated when app features, storage behavior, or network behavior changes. Updates will be posted on this page.

  • 앱이 표시하는 경로, 파일명, 로그 내용에는 개인 정보가 포함될 수 있으므로 공유 전 사용자가 검토해야 합니다.

4. 로컬 저장 정보

PC Bareun은 앱 설정, 언어와 테마 설정, 최근 실행 상태, 기능 실행 결과, 오류 진단 로그를 사용자의 기기에 저장할 수 있습니다. 로컬 로그에는 기능 이름, 오류 메시지, 파일 경로 또는 파일명이 포함될 수 있습니다.

기본 로그 위치: %LOCALAPPDATA%\com.pcbareun.app\logs\

이 정보는 앱 문제를 진단하고 사용자가 요청한 기능을 안정적으로 제공하기 위해 사용됩니다.

5. 네트워크 요청

PC Bareun은 정리 결과, 파일 목록, 시스템 진단 결과를 자동으로 외부 서버에 업로드하지 않습니다. 다만 다음 작업에서는 기능 수행을 위해 네트워크 요청이 발생할 수 있습니다.

  • 사용자가 웹사이트, 지원 페이지, GitHub 링크를 여는 경우
  • 사용자가 네트워크 진단 또는 속도 테스트 기능을 실행하는 경우
  • Microsoft Store가 아닌 직접 배포판에서 사용자가 업데이트 확인 기능을 실행하는 경우

Microsoft Store 버전의 앱 업데이트는 Microsoft Store 업데이트 체계를 따릅니다.

6. 제3자 제공 및 판매

PC Bareun은 개인정보를 판매하지 않습니다. 또한 앱에서 처리한 파일 내용, 정리 결과, 진단 결과를 광고 회사나 분석 서비스에 제공하지 않습니다. 사용자가 GitHub Issues에 문의를 남기거나 로그를 직접 첨부하는 경우, 해당 내용은 사용자가 선택하여 공유한 정보로 처리됩니다.

プライバシーポリシー (日本語)

PC Bareun は、Windows PC のクリーンアップ、診断、ローカル管理を支援するデスクトップアプリです。クリーンアップ結果や診断情報は、基本的にユーザーの端末内で処理されます。 PC Bareun は広告 SDK、分析 SDK、ユーザー追跡 SDK を含まず、個人情報を販売しません。

端末上で処理される情報

PC Bareun は、ユーザーが選択した機能を提供するために、ファイル名、パス、サイズ、更新日、フォルダー構造、一時ファイル、ブラウザーキャッシュ、最近使用した項目、インストール済みプログラム、スタートアップ項目、Windows サービス、タスクスケジューラー項目、ディスク状態、メモリ、ハードウェア、ネットワーク状態、BSOD イベント、エラーメッセージ、開発ツールのキャッシュ、AI モデルファイル、環境変数、PATH 情報などを端末上で確認する場合があります。

ネットワーク通信

PC Bareun は、ファイルの内容、クリーンアップ結果、診断結果を自動的に外部サーバーへアップロードしません。ユーザーが Web サイト、サポートページ、GitHub リンクを開く場合、ネットワーク診断や速度テストを実行する場合、または Microsoft Store 版ではない直接配布版で更新確認を実行する場合に、必要なネットワーク要求が発生することがあります。Microsoft Store 版の更新は Microsoft Store を通じて提供されます。

ローカルログとユーザーの管理

PC Bareun は、設定やトラブルシューティング用ログをユーザーの端末に保存する場合があります。ログにはローカルパス、ファイル名、機能名、エラーメッセージが含まれることがあります。公開された場所へ共有する前に内容を確認してください。

既定のログ保存場所: %LOCALAPPDATA%\com.pcbareun.app\logs\
