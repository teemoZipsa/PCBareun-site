지원 문의

PC Bareun 사용 중 오류, 설치 문제, 기능 제안, 개인정보 관련 문의가 있으면 GitHub Issues에 내용을 남겨 주세요.

지원 접수: https://github.com/teemoZipsa/PCBareun/issues

문의할 때 포함하면 좋은 정보

  • PC Bareun 버전
  • Windows 버전과 빌드 번호
  • 문제가 발생한 기능 이름
  • 오류 메시지 또는 스크린샷
  • 문제가 반복되는 단계

로컬 로그 위치

오류 진단에 필요한 경우 아래 위치의 로그 파일을 확인할 수 있습니다.

%LOCALAPPDATA%\com.pcbareun.app\logs\

로그에 개인 경로 또는 파일명이 포함될 수 있으므로 공유 전 내용을 확인해 주세요.

Support

For bugs, installation issues, feature requests, or privacy questions, submit a GitHub Issue with the relevant details.

Support URL: https://github.com/teemoZipsa/PCBareun/issues

Helpful Details

  • PC Bareun version
  • Windows version and build number
  • Feature where the issue occurred
  • Error message or screenshot
  • Steps to reproduce the issue

Local Logs

For troubleshooting, local logs may be available at:

%LOCALAPPDATA%\com.pcbareun.app\logs\

Logs may include local paths or file names, so review them before sharing.
